The launch.R code is used to initialize the every script, TensorFlow and Keras packages need to loaded with 
versions that are compatible for Python. This study utilizes Python 3.10 with TensorFlow version 2.10.1.

The DMTest.R script runs the Diebold-Mariano test to compare forecast accuracy between models, and can be adapted to 
test any combination of saved .rds forecast files generated by the provided LSTM and GRU scripts.
